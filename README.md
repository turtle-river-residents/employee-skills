## How to run

1. How to build the API Project
    ```shell
    cd ./employee-skills-api
    sh ./gradlew build
    ```
2. How to build the WEB Project
   ```shell
   cd ./employee-skills-web
   yarn build
   ```
3. How to build the Docker and run
   ```shell
   docker-compose build
   dokcer-compose up -d
   ```
